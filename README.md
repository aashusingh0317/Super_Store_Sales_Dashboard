# Super_Store_Sales_Dashboard

## Project Overview

This project analyzes Super Store sales data using Power BI to gain insights into business performance, sales trends, and customer purchasing patterns. The interactive dashboard helps track key performance indicators and supports data-driven decision making.

## Tools Used

* Power BI
* Excel / CSV Dataset
* Data Visualization

## Key Features

* KPI cards for Total Sales, Profit, Orders, and Average Shipping Days
* Sales and Profit analysis by State using map visualization
* Sales analysis by Category and Sub-Category
* Sales distribution by Ship Mode
* Sales analysis by Payment Mode
* Customer segmentation analysis
* Regional sales performance analysis
* Monthly Sales and Profit trend analysis

## Key Insights

* The **West region** generates the highest sales compared to other regions.
* **Standard Class shipping mode** contributes the highest number of sales.
* **Office Supplies category** generates the highest revenue.
* **Consumer segment** contributes the largest share of sales.

## Conclusion

This dashboard helps businesses monitor sales performance, identify profitable product categories, and understand customer behavior to improve business strategies.
